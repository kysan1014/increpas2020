함수 (Function, Method)
=> 한가지 작업을 실행하기 위해서 만들어진 가장 작은 단위의 프로그램

	프로그램이란?
		이런 함수들을 모아 원하는 목적을 이루도록 구성하는 것
	
	예 )
	
		charAt()
			: 한 글자를 알아내는 작업을 실행하기 위해 만들어진 함수
			
		random()
			: 난수 하나를 발생시키는 작업을 실행하기 위해 만들어진 함수
			
***
참고

	1.함수는 누군가 호출을 해야 실행된다.
		즉, 스스로 실행되는 함수는 없다.
		
	2. 함수는 실행이 완료되면 자신을 호출한 곳으로 실행 순서를 넘겨준다.
	
		class Test {
			int no = 0;
			String str = "홍길동";
			
			public static void abc() {
				System.out.println("비는 조금만... 제발...!");
			}
		
			public static void main(String[] args) {
				abc();
				System.out.println("여기는 메인...");
			}
		
		}

함수 만드는 방법

	형식 ) 
	
		[ 접근 지정자 ] [ 속성 ] 반환타입 함수이름(매개 변수 리스트) {
			실행 내용...
		}
		
		1. 함수 이름
			- 개발자가 지정하는 사용자 정의어...
			
		2. 매개변수 리스트
			- 변수 선언 형태로 만드는 것
			- 필요한만큼 만들어도 됨
			
			예 )
			
				abc(int a, int b) {
				
				}
				
				실행할 때,
				
				abc(10, 9);
				
		목적 )
		
			작성하는 함수가 원하는 작업을 실행하기 위해서 반드시 필요한 데이터가 있을 것이다.
			이 데이터를 기억할 준비를 하는 것
			
			예 )
			
				charAt(int pos) -> 원하는 위치의 문자를 알려주는 함수
									이 함수가 원하는 작업을 충실히 수행하기 위해서는
									위치 값이 반드시 필요하다.
		
		3. 반환타입
			- 함수는 한 가지 목적을 달성하기 위한 가장 작은 단위의 프로그램이다
				- 함수를 실행한다는 말은 그 목적을 달성한다는 것과 동일하다.
					그러면 그 달성된 목적에 따라 결과가 존재해야 할 것이다.
			
			반환값타입
				- 함수에게 어떠한 작업을 시켰을 때,
					그 함수가 수행한 작업의 결과를 알려주는 것이다.
					- 결과는 경우에 따라서 달라질 수 있으므로,
						결과값의 데이터 종류만 선언하면 된다.
						
			참고 )
			
				함수의 실행결과가 반드시 존재하는 것은 아니다.
				예를 들어
					System.out.println();
					이 함수의 기능은 콘솔화면에 문자열을 출력하는 기능만 가지고 있다.
					콘솔 화면에 문자열을 출력하게 되면 함수의 목적은 모두 이루어진 것
			
			참고 ) 
			
				함수는 가장 작은 기능의 단위이고,
				반환값은 한 개이거나 없다.
				반환 값이 두 개 이상인 경우는 존재하지 않는다.
			
			결론 ) 
			
			   함수는 네 가지 형태를 벗어나지 않는다.
			   
				   1. void abc() {}
				   
				   2. int abc() {}
				   
				   3. void abc(int a) {}
				   
				   4. int abc(int a) {}
				   
			return의 의미
				- 함수 내에서 사용할 수 있는 명령
				
				형식 ) 
				
					1. return; - 반환값이 없는 경우
					
					2. return 데이터; - 반환값이 있는 경우
					
				참고 ) 
				
					반환타입이 void인 경우에는 return을 생략해도 된다.

함수 사용법

	- 함수는 반드시 호출해야 사용할 수 있다.
		위의 내용은 함수의 기능을 정의하는 것일 뿐이다.
		정의 내용이 스스로 실행되는 경우는 없다.
		
		따라서 함수를 사용한다는 것은 함수를 호출한다는 것이다.
		
		형식 )
		
			함수이름([데이터], [데이터] ...);
			
			데이터란?
				함수는 그 함수가 실행하기 위해 필요한 데이터를 받은 준비를 하고 있다(매개변수 리스트)
				따라서 그 함수에게 작업을 시키기 위해서는
				필요한 데이터를 반드시 입력해야 한다.
				
					- 결론적으로 데이터(Parameter)란?
						
						함수가 준비하고 있는 맥개변수 리스트에 기억할 데이터를 제공하는 것이다.
				
					- 결론
					
						데이터는 그 함수가 준비한 매개변수의 개수와 타입이 반드시 일치해야 함
